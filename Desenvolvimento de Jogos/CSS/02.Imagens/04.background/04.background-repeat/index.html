background-repeat:repeat;
background-repeat:repeat-x; Determina que as imagens só devem ser repetidas dentro do container no eixo x.
background-repeat:repeat-y; Determina que as imagens só devem ser repetidas dentro do container no eixo y.
background-repeat:space; Ira se repetir na sua totalidade e por todo container sem que haja quebras na imagem, então o space determina qual o tamanho ideal para comportar as repetições, com espaçamento entre ela e sem que seja necessario corta-las.
background-repeat:round; Sem cortar as imagens que sera repetida, realiza o redimensionamento para que se repita de forma proporcional.
background-repeat:no-repeat;
background-repeat: space round; Usando mais do que uma forma no mesma imagem, basta definir o eixo x e y.
