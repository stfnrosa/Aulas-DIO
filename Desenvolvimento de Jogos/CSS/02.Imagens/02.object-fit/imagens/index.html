Propriedades object-fit

especifica como uma imagem deve ser redimensinada para se ajustar ao container. Para que o ajuste seja feito, se será necssario que ela seja redimensionada ou que as proporções seejam mantidas

object-fit:fill; (padrão): redimensionado para o tamanho do container IGNORANDO A PROPORÇÃO da imagem.
object-fit:contain; (não redimensiona): RESPEITA A PROPORÇÃO da imagem para caber no container mas não é redimensionada.
object-fit:cover; (não redimensiona): preenche todo o conteiner sem ser redimensionada e RESPEITA A PROPORÇÃO, caso seja mais extensa que o proprio container será cortada.
object-fit:none; (não redimensiona): mantem o tamanho original da imagem e ignorando o tamanho(altura/largura) do elemento pai, mas quando o container for menor, pode ser cortada.
object-fit:scale-down; (não redimensiona): escolhe entre object-fit:contain; e object-fit:none; o que resultara em um tamanho menor de objeto concreto.
